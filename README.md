# Supermart Sales Analysis

## Overview

This project is a basic data analysis of a Supermart sales dataset using Python.

In this project, I worked on understanding the data and finding simple insights related to sales, profit, and products. I also practiced data cleaning and visualization.

---

## What I Did

* Cleaned the dataset (handled unnecessary columns and checked missing values)
* Converted date columns into proper format
* Created new columns like month and year
* Analyzed sales, profit, and quantity
* Found top products and categories
* Compared sales across different years and regions
* Created some basic visualizations

---

## Tools Used

* Python
* Pandas
* Matplotlib

This project is part of my learning journey in data analysis.
